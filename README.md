# 💸 Personal Expense Analyzer

A data-driven web application built using Streamlit to track, analyze, and visualize personal expenses.

## 🚀 Features
- Upload CSV or manually add expenses
- Interactive dashboards and charts
- Budget tracking and alerts
- Expense prediction using Machine Learning
- User authentication (SQLite)

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas, NumPy
- Plotly
- Scikit-learn
- SQLite

## ▶️ Run Locally
```bash
python -m streamlit run app.py
